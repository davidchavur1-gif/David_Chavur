Resumen de la Unidad 4

Conversión y Adaptación de Documentos XML

📄 XML  ¿Qué es XML?

eXtensible Markup Language — lenguaje de marcado para almacenar y transportar datos de forma estructurada.

  • Similar a HTML pero sin etiquetas predefinidas — tú defines las tuyas
  • Pensado para el intercambio de datos entre sistemas y plataformas
  • Garantiza portabilidad, escalabilidad y validación de datos

⚙️ XML  XMLHttpRequest

Objeto JavaScript para hacer peticiones HTTP sin recargar la página.

  • Peticiones GET y POST
  • Modo síncrono (false) o asíncrono (true)
  • Procesa respuestas en XML, JSON, texto o HTML

Flujo de carga de un XML:

  • 1. new XMLHttpRequest() — crear objeto XHR
  • 2. xhr.open("GET","archivo.xml",false) — abrir conexión
  • 3. xhr.send() — enviar solicitud
  • 4. xhr.status === 200 — comprobar respuesta
  • 5. xhr.responseXML — convertir a XML
  • 6. getElementsByTagName("pelicula") — extraer datos

🔄 XSL  XSLT

eXtensible Stylesheet Language Transformations — transforma XML en HTML, texto plano u otros formatos.

  • Extrae, modifica y presenta datos sin alterar el XML original
  • Genera etiquetas HTML (<table>, <div>, etc.) dinámicamente
  • Para vincular un XSL al XML: <?xml-stylesheet type="text/xsl" href="archivo.xsl"?>
  • No todos los navegadores modernos admiten XSLT nativo — puede requerir servidor local o JavaScript

✅ Resumen clave

  • XPath facilita la selección eficiente de datos en XML
  • XPath permite manipular y transformar datos XML de manera sencilla
  • XSLT convierte XML en HTML, texto plano o incluso otros XML
  • XSLT estructura y filtra información según necesidades específicas
